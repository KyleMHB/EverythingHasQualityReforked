[h1]Everything Has Quality Reforked[/h1]

[i]A fork of a fork that brings quality to far more things in RimWorld, with configurable categories, individual item control, search, and better settings performance for large modpacks.[/i]

[h2]What It Does[/h2]
This mod expands RimWorld's quality system so more items can have quality, while letting you decide [b]where[/b] quality applies.

You can enable quality for:
[list]
[*]Buildings
[*]Materials
[*]Ingredients
[*]Meals
[*]Drugs
[*]Medicine
[*]Apparel
[*]Weapons
[*]Shells
[*]Other manufactured items
[/list]

You can also:
[list]
[*]Use category-wide settings
[*]Override behavior for individual defs
[*]Search long item lists in the settings menu
[*]Set minimum and maximum quality ranges by item group
[*]Optionally factor in materials, work tables, skills, and inspirations
[/list]

[h2]Why This Refork Exists[/h2]
This refork keeps the original mod's core idea intact, while improving usability and stability for modern 1.6 modpacks.

[b]This refork adds:[/b]
[list]
[*]Search for long settings lists
[*]Better performance in large settings lists
[*]Translation support for the custom inspirations used by this fork
[*]Bug fixes, stability work, and compatibility fixes for RimWorld 1.6
[/list]

[h2]Important Notes[/h2]
This mod adds quality to more things, but it does [b]not[/b] by itself make quality affect every stat on those things.

If you want quality to affect things like durability or broader stat scaling, you will usually want a companion mod such as [b]Quality Expanded[/b].

This mod is especially useful as a configurable quality-enabler for larger modded setups.
